HTML

HyperText Markup Language

Bahasa dasar untuk membuat struktur dari sebuah halaman website.
HTML menentukan elemen apa saja yang muncul di layar, seperti teks, gambar, link, atau tabel.
HTML bukan bahasa pemrograman, tapi bahasa markup untuk menyusun konten.

Struktur Dasar HTML

Cara Menggunakan HTML

1. Buat file baru bernama index.html.

2. Tulis struktur dasar seperti di atas.

3. Simpan file.

4. Buka file tersebut di browser & website sederhana akan tampil.
